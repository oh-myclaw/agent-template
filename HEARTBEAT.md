# HEARTBEAT.md — Periodic Tasks

Use this file for tasks that should be checked every heartbeat cycle.

Each task must be a bullet line starting with `- `.
Optional tags:
- `[high]` or `[low]` for priority (default: medium)
- `[paused]` or `[completed]` for status

Examples:
- [high] Check my inbox for urgent messages
- Review my calendar for tomorrow
- [low|paused] Check package tracking updates
